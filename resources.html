How to Set Up and Run Your Basic Bot

1. Install Node.js

Make sure you have Node.js installed on your computer. You can download it from the official website: https://nodejs.org/en

2. Set Up the Project

  1. Create a New Folder: Choose a location on your computer and create a folder for your bot project. For example, name it MyDiscordBot.
  2. Navigate to the Folder: Open a terminal and navigate to the folder using the cd command: 
    cd path/to/MyDiscordBot
  3. Initialize the Project: Run the following command to create a package.json file: 
    npm init -y

3. Install Dependencies

Run the following command in the terminal to install the required packages:

npm install discord.js dotenv

4. Create Necessary Files

  1. Create the Bot File: In your project folder, create a file called index.js and paste the bot's code into it.
  2. Create a .env File: In the same folder, create a file named .env and add: 
    TOKEN=YOUR_BOT_TOKEN
    Replace YOUR_BOT_TOKEN with your bot token from the Discord Developer Portal.
  3. Create the Folders: Run the bot once to automatically create the commands and events folders.

5. Run the Bot

Start the bot using this command:

node index.js

You should see a startup message in the console and a message indicating the bot is logged in.

6. Add Commands and Events

Use the commands and events folders for modular bot functionality:

  • Commands: Create a file in the commands folder, e.g., ping.js: 
    module.exports = {
            data: { name: 'ping' },
            async execute(interaction) {
                await interaction.reply('Pong!');
            },
        };
  • Events: Create a file in the events folder, e.g., ready.js: 
    module.exports = {
            name: 'ready',
            once: true,
            execute(client) {
                console.log(`${client.user.tag} is ready!`);
            },
        };

7. Invite Your Bot to a Server

  1. Go to the Discord Developer Portal.
  2. Open your bot's application and navigate to OAuth2 > URL Generator.
  3. Select bot under Scopes and choose necessary permissions under Bot Permissions.
  4. Copy the generated URL, paste it into your browser, and invite the bot to your server.

8. Test Your Bot

Start the bot again and test its commands and events in Discord.
